基本按钮

基本样式：btn ，颜色效果基本样式：btn-default，btn-primary等
✔ 大小：需要改变大小加样式：btn-lg，btn-sm，btn-xs



✔圆角/圆形：圆角样式： btn-rounded，圆形样式：btn-circle
✔hover变色：如果需要鼠标移动到按钮变色，叠加加btn-hover-* 样式即可，如btn-hover-warning
✔click变色：如果在点击的时候变色，叠加加btn-active-* 样式即可，如btn-active-success
✔disabed按钮：按钮失效，在按钮元素加属性disabled="disabled"即可
✔图标按钮：1：按钮元素加样式 btn-icon 2：按钮内容加入图标元素标签即可
✔图标+文字按钮：1：按钮元素加样式 btn-labeled 2：按钮内容加入图标元素标签 3：正常写按钮label文本
✔锁定按钮：按钮元素加样式 btn-block
